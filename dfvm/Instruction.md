# Data flow virtual machine instruction set

* Data store model
  * Register (temporary data)
  * Local variables
  * Argument stack (flow data)


## MOV -- Move data

    MOV src,dst

## ADD

    ADD op1,op2

## SUB

    SUB op1,op2

## MUL

    MUL op1,op2

## DIV

    DIV dividend,divisor,quotient[,remainder]

## YIELD

    YIELD
